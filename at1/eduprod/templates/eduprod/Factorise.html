Factorising

To solve quadratic equations through factorization, follow these steps:

  1. Identify the Quadratic Equation:
    A quadratic equation is in the form ax^2 + bx + c = 0,

  2. Factor the Equation:
    Look for two numbers that multiply to ac and add to b.

  3. Set Each Factor Equal to Zero:
    Once you have the factors, set each factor equal to zero and solve for x.
    This gives you the solutions to the quadratic equation.

  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

Here is an Example.

Solve the quadratic equation: x2 - 7x + 12 = 0

  1. Identify the Quadratic Equation:
    x2 - 7x + 12 = 0.
    a = 1,
    b = -8,
    c = 12.

  2. Factor the Equation:
    Look for two numbers that multiply to ac and add to b.
    -3 * -4 = 12 (ac),
    -3 + -4 = -7 (b),
    Hence, the Factor form would be (x - 3)(x - 4) = 0

  3. Set Each Factor Equal to Zero:
    Solve for x through the null factor law which is [ab = 0] either a or b is 0.

    x - 3 = 0.
    x = 3

    x - 4 = 0.
    x = 4

  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

    (3)2 - 7 * (3) + 12 = 0,
    9 - 21 + 12 = 0,
    0 = 0.

    (4)2 - 7 * (4) + 12 = 0,
    16 - 28 + 12 = 0,
    0 = 0.

Hence, the solutions to the quadratic equation x2 - 7x + 12 = 0 are x = 2 and x = 6.

Solve the factorise and solve the equation: 4x2 - 9

  1. Identify the Quadratic Equation:
    4x2 - 9.
    a = 4,
    b = 0,
    c = -9.

  2. Factor the Equation:
    Look for two numbers that multiply to ac and add to b.
    However there are no solutions via this method so we will have to have use the differences of square to factorise our expression.
    Using a2 - b2 = (a + b)(a - b):

    a2 = (2x)2,
    a = 2x,

    b2 = (3)2,
    b = 3,

    Hence, the Factor form would be (2x + 3)(2x - 3).

  3. Set Each Factor Equal to Zero:
    (2x + 3)(2x - 3) = 0.

    Solve for x through the null factor law which is [ab = 0] either a or b is 0.

    2x - 3 = 0.
    2x = 3
    x = 3⁄2
    2x + 3 = 0.
    2x = -3
    x = -3⁄2

  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

    4 * (3⁄2)2 - 9 = 0,
    4 * (9⁄4) - 9 = 0,
    9 - 9 = 0,
    0 = 0.

    4 * (-3⁄2)2 - 9 = 0,
    4 * (9⁄4) - 9 = 0,
    9 - 9 = 0,
    0 = 0.
