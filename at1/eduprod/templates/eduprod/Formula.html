Quadratic Formula

To solve quadratic equations through Quadratic Formula, follow these steps:

  1. Identify the Quadratic Equation:
    A quadratic equation is in the form ax2 + bx + c = 0,

  2. Substitute the Terms:
    Replace the constants with the numbers. x = -b ± √b2 - 4ac2a

  3. Solve for x:
    Simplify and Solve for x.

  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

Here is an Example.

Solve the quadratic equation: x2 + x - 1 = 0

  1. Identify the Quadratic Equation:
    x2 - x - 1 = 0
    a = 1,
    b = -1,
    c = -1.

  2. Substitute the Terms:
    x = -b ± √b2 - 4ac2a
    x = -(-1) ± √(-1)2 - 4 * (1) * (-1)2 * (1)

  3. Solve for x:
    x = 1 ± √52
    Hence, x = 1 ± √52
  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

    (1 + √52)2 - (1 + √52) - 1 = 0,
    (1 + √5)222 - 1 + √52 - 1 = 0,
    1 + 2√5 + 54 - 1 + √52 - 1 = 0,
    6 + 2√54 - 1 + √52 - 1 = 0,
    3 + √52 - 1 + √52 - 1 = 0,
    22 - 1 = 0,
    1 - 1 = 0,
    0 = 0.

    (1 - √52)2 - (1 - √52) - 1 = 0,
    (1 - √5)222 - 1 - √52 - 1 = 0,
    1 - 2√5 + 54 - 1 - √52 - 1 = 0,
    6 - 2√54 - 1 - √52 - 1 = 0,
    3 - √52 - 1 - √52 - 1 = 0,
    22 - 1 = 0,
    1 - 1 = 0,
    0 = 0.
