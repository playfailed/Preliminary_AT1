Complete the Square

To solve quadratic equations through Completing the Square, follow these steps:

  1. Identify the Quadratic Equation:
    A quadratic equation is in the form ax^2 + bx + c = 0,

  2. Add the term to complete the square:
    If there is a c term within the quadratic move it to ther other side.
    Add (b2)2 to both sides of the equation.

  3. Factor the perfect square trinomial:
    Once you have a perfect square trinomial, factor it to isolate the variable term.
    Then solve for x. Rememeber ± when square terms.

  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

Here is an Example.

Solve the quadratic equation: x2 + 8x + 5 = 0

  1. Identify the Quadratic Equation:
    x2 + 8x + 5 = 0

  2. Add the term to complete the square:
    Move the c term to the otherside
    x2 + 8x = -5
    Adding (22)2 to both sides of the equation.
    x2 + 8x + (82)2 = -5 + (82)2
    x2 + 8x + 644 = -5 +644
    x2 + 8x + 16 = -5 + 16
    x2 + 8x + 16 = 11
    Hence, we have a perfect square trinomial, (x + 4)2 = 11

  3. Solve for x:
    (x + 4)2 = 11
    x + 4 = ± √11
    x = -4 ± √11
    Hence, x = -4 - √11 and x = -4 + √11

  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

    (-4 -√11)2 + 8 * (-4 - √11) + 12 = 0,
    16 + 8√11 + 11 + -32 - 8√11 + 12 = 0,
    16 + 11 + 5 -32 = 0
    0 = 0.

    (-4 + √11)2 + 8 * (-4 + √11) + 12 = 0,
    16 - 8√11 + 11 + -32 + 8√11 + 12 = 0,
    16 + 11 + 5 -32 = 0
    0 = 0.

Hence, the solutions to the quadratic equation x2 - 7x + 12 = 0 are x = -4 ± √11

Here is another Example.

Solve the quadratic equation: 2x2 + 3x + 9 = 0

  1. Identify the Quadratic Equation:
    2x2 + 3x + 9 = 0

  2. Add the term to complete the square:
    Move the c term to the otherside
    2x2 + 3x = -9
    Since this quadratic is non monic you must divide the whole equation by a.
    x2 + 3x2 = -92
    Adding (b2)2 to both sides of the equation.
    x2 + 3x2 + (34)2 = (34)2 -92
    x2 + 3x2 + 916 = 916 -92
    x2 + 3x2 + 916 = 916 -7216
    x2 + 3x2 + 916 = -6316
    Hence, we have a perfect square trinomial, (x + 34)2 = -6316

  3. Solve for x:
    There's no real solutions since we can't square root a negative number.
    Here's the working out if you want to know it.
    x + 34 = ±√-63√16
    x + 34 = ±√-634
    x + 34 = ±3√-74
    x = - 3 ± 3√-74

  4. Verify the Solutions:
    Substitute each solution back into the original equation to verify that it is indeed a solution.

    2 * ((- 3 + 3√-7)242) + 3 * (- 3 + 3√-74) + 9 = 0,
    2 * ((- 3 + 3√-7)216) + 3 * (- 3 + 3√-74) + 9 = 0,
    2 * (9 - 18√-7 - 6316) + 3 * (- 3 + 3√-74) + 9 = 0,
    2 * (-18√-7 - 5416) + 3 * (- 3 + 3√-74) + 9 = 0,
    2 * (-9√-7 - 278) + 3 * (- 3 + 3√-74) + 9 = 0,
    -9√-7 - 274 + 3 * (- 3 + 3√-74) + 9 = 0,
    -9√-7 - 274 + - 9 + 9√-74 + 9 = 0,
    - 364 + 9 = 0,
    -9 + 9 = 0,
    0 = 0.

    2 * ((- 3 - 3√-7)242) + 3 * (- 3 - 3√-74) + 9 = 0,
    2 * ((- 3 - 3√-7)216) + 3 * (- 3 - 3√-74) + 9 = 0,
    2 * (9 + 18√-7 - 6316) + 3 * (- 3 - 3√-74) + 9 = 0,
    2 * (18√-7 - 5416) + 3 * (- 3 - 3√-74) + 9 = 0,
    2 * (9-7 - 278) + 3 * (- 3 - 3√-74) + 9 = 0,
    9√-7 - 274 + 3 * (- 3 - 3√-74) + 9 = 0,
    9√-7 - 274 + - 9 - 9√-74 + 9 = 0,
    - 364 + 9 = 0,
    -9 + 9 = 0,
    0 = 0.

Hence, there are no real solutions.

The imaginary solutions to the quadratic equation 2x2 + 3x + 9 = 0 are x = - 3 ± 3√-74
